--Instruction---

--Name of the files--

(1) airline_train: Pre-modified train data for the implementation in Matlab
(2) airline_test: Pre-modified test data for the implementation in Matlab
(3) Coursework_EDA_and_Preprocessing: EDA and preprocessing done by Python 
(4) Naive_Bayes_Train: 10 Fold cross-validation Naive Bayes model without hyper parameter tuning
(5) Naive_Bayes_HP_tuning: 10 Fold Grid Search to tune hyper parameter
(6) Naive_Bayes_Train_With_HPtuning: 10 Fold cross-validation Naive Bayes model for train data with the most optimised hyper parameter 
(7) Naive_Bayes_Test: Predictive model for test data with the most optimised hyper parameter
(8) Random_Forest_Train: 10 Fold cross-validation Random Forest model without hyper parameter tuning
(9) Random_Forest_HP_tuning: 10 Fold Grid Search to tune hyper parameter
(10) Random_Forest_Visualisations: Visualisation for the trend of explored hyper parameter. 
(11) Random_Forest_Train_With_HPtuning: 10 Fold cross-validation Random Forest model for train data with the most optimised hyper parameter 
(12) Random_Forest_Test: Predictive model for test data with the most optimised hyper parameter
